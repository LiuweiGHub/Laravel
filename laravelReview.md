# laravel回顾

## 一、laravel环境搭建

- 要求：PHP>=5.6.4

- 且PHP需安装5个扩展

  - OpenSSL PHP Extension
  - PDO PHP Extension
  - Mbstring PHP Extension
  - Tokenizer PHP Extension
  - XML PHP Extension

- 需安装composer[laravel利用composer管理其依赖文件]

  ```shell
  //安装laravel指令
  composer global require "Laravel/installer"
  ```
